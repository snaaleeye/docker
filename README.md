## Docker



`docker run -d -p 80:80 nginx`

`docker ps -a` 

`docker start `
`docker stop `

To get inside container run this command
`docker exec -it 4714565e9cbf bash`

`alias docker="winpty docker"`

`apt-get update -y`
`apt-get install nano`
